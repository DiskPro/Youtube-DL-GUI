Youtube-DL GUI

English?

Este é um programa para Windows. Não vai rodar em um celular. Desculpe!

Sobre

 Youtube-DL GUI é uma Interface Gráfica de Usuário para o programa youtube-dl criada por mim, Guilherme Francisco de Freitas. Foi programada em C# e ainda recebe atualizações frequents. Além de gratuita, é open source e continuará desse modo para sempre.

Recursos

  • Gratuito!
  • Open source
  • Preview de thumb
  • Importa listas de vídeos
  • Compatível com centenas de serviços de streaming
  • Leve
  • Fácil de usar
  • Aceita vários formatos de vídeo e áudio
  • Atualizações automáticas
  • Criação de listas de vídeos
